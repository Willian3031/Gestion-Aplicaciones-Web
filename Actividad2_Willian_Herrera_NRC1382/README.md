# Proyecto Web - Registro y Funcionalidades

Este proyecto es una aplicación web interactiva que permite:
1. **Registrar usuarios:** Captura datos personales y los muestra en una nueva ventana.
2. **Realizar funcionalidades lógicas:**
   - Comparación de números.
   - Determinar el mes correspondiente a un número ingresado.
   - Gestión dinámica de arrays.

## Estructura del proyecto

El proyecto está organizado en las siguientes carpetas:

- **`css/`**: Contiene los estilos para las diferentes páginas:
  - `general.css`
  - `funcionalidades.css`
  - `inicio.css`
  - `registro.css`

- **`js/`**: Contiene los scripts para manejar las funcionalidades de la aplicación:
  - `funcionalidades.js`
  - `registro.js`

- **Raíz del proyecto**:
  - `index.html`: Página de inicio.
  - `funcionalidades.html`: Funcionalidades lógicas.
  - `registro.html`: Registro de usuarios.

## Instrucciones para ejecutar

1. **Abrir la página principal:**
   - Navega hasta el archivo `index.html` en el directorio raíz y ábrelo en tu navegador.

2. **Navegar por la aplicación:**
   - Usa el menú de navegación en la parte superior para moverte entre las siguientes páginas:
     - **Inicio**: Información introductoria.
     - **Funcionalidades**: Incluye comparación de números, selección de meses y manejo dinámico de arrays.
     - **Registro**: Formulario para registrar usuarios, cuyos datos se mostrarán en una nueva ventana.

3. **Registrar usuarios:**
   - Ingresa los datos requeridos en el formulario en la página `registro.html`.
   - Presiona el botón "Registrar" para enviar los datos.
   - Los datos registrados aparecerán en una nueva ventana.

## Notas importantes

- Todos los scripts han sido implementados utilizando funciones flecha (`arrow functions`) en JavaScript, cumpliendo con los requisitos del proyecto.
- El almacenamiento temporal de datos utiliza `localStorage` para manejar la persistencia mientras la página está abierta.

## Requisitos

- Un navegador web moderno (Google Chrome, Firefox, Edge, etc.).
- El proyecto no requiere instalación adicional ni un servidor externo para ejecutarse.
